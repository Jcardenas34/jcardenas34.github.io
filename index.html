Projects

Here is a link to my personal projects on Github demonstrating my expertise in Python, Deep Learning, and Statistics

Fraud Detection on open source Credit/Debit Card transactions: In this repository I perform a statistical analysis on open source credit card data to detect instances of fraud using standard statistical methods as well a variational auto-encoder and standard auto encoder.

Understanding Infectious diseases in California: I use a data set provided by the state of California to derive insights on the most prevalent diseases in the most populous counties, as well as project case rates based on historical data using various regression models.

Predicting Failure, NASA TurboFan Engine: My most recent project involves predicting the remaining useful life of jet engines through on board sensor data by using a Hidden Markov Model as well as other ML techniques, this one is a work in progress!
